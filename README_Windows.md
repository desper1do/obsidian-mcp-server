# MCP Сервер для Базы Знаний (Windows)

Этот локальный сервер позволяет нейросетям безопасно читать, искать и редактировать вашу базу знаний Obsidian прямо на вашем компьютере.

## 🚀 Установка и настройка (Один раз)

1. Распакуйте эту папку в любое место на вашем компьютере.
2. Дважды кликните по файлу `Управление_сервером_Windows.bat`. Скрипт сам создаст виртуальное окружение и скачает нужные библиотеки.
3. В открывшемся черном окне нажмите `1` и перетащите туда папку с вашей Базой Знаний (затем нажмите Enter). Вы можете добавить так несколько разных папок.
4. Нажмите `3`, чтобы сгенерировать конфигурацию для подключения.

## 🔌 Как подключить к расширению Claude в VS Code
1. Откройте VS Code и вызовите терминал (верхнее меню: `Terminal` -> `New Terminal`).
2. Вставьте в терминал эту команду и нажмите Enter:
   `code %APPDATA%\Claude\settings.json`
3. У вас откроется файл настроек Клода. Скопируйте JSON-код из пункта 3 нашего меню и вставьте в этот файл.
   *(Если там уже есть другие настройки, аккуратно добавьте блок `"mcpServers"` внутрь главных скобок `{}`).*
4. Сохраните файл (`Ctrl + S`).
5. Нажмите `Ctrl+Shift+P`, введите **Developer: Reload Window** и нажмите Enter. Всё готово!

## 🔌 Как подключить к консольному Claude Code (CLI)
Если вы используете Claude прямо в терминале, просто выберите пункт `4` в меню `Управление_сервером_Windows.bat`. Скрипт сам всё подключит!

## 🧠 Как общаться с ИИ (Примеры промптов)
Просто общайтесь с ИИ как с живым ассистентом, он сам подберет нужные инструменты для работы с вашими файлами:
* *"Я оставил новые мысли в черновике `заметки_с_планерки.md`. Прочитай их и разнеси эти изменения по нужным чистовым файлам базы знаний."*
* *"Найди все файлы, где упоминается преддипломная практика, и выпиши мне все дедлайны."*

**Примечание:** Перед любым изменением файлов сервер автоматически создает резервную копию оригинала в скрытой папке `.backups`. Ваши данные в полной безопасности.
