# MCP Сервер для Базы Знаний (macOS / Linux)

Этот локальный сервер позволяет нейросетям безопасно читать, искать и редактировать вашу базу знаний Obsidian прямо на вашем Mac.

## 🚀 Установка и настройка (Один раз)

1. Распакуйте эту папку в удобное место (например, в `Documents`).
2. ВАЖНО: При первом запуске нажмите по файлу `Управление_сервером_MacOS.command` **правой кнопкой мыши** (или двумя пальцами по тачпаду) и выберите **"Открыть"** (Open). Система может спросить, доверяете ли вы файлу — подтвердите запуск. В будущем можно будет просто кликать дважды.
3. Скрипт сам создаст виртуальное окружение в терминале и скачает библиотеки.
4. В открывшемся меню нажмите `1` и перетащите в окно терминала папку с вашей Базой Знаний (нажмите Enter).
5. Нажмите `3`, чтобы сгенерировать конфигурацию для подключения.

## 🔌 Как подключить к расширению Claude в VS Code
1. Откройте VS Code и вызовите терминал (верхнее меню: `Terminal` -> `New Terminal`).
2. Вставьте в терминал эту команду и нажмите Enter:
   `code ~/.claude/settings.json`
3. Откроется файл настроек Клода. Скопируйте JSON-код из пункта 3 нашего меню и вставьте в этот файл.
   *(Если там уже есть другие настройки, аккуратно добавьте блок `"mcpServers"` внутрь главных скобок `{}`).*
4. Сохраните файл (`Cmd + S`).
5. Нажмите `Cmd+Shift+P`, введите **Developer: Reload Window** и нажмите Enter. Всё готово!

## 🔌 Как подключить к консольному Claude Code (CLI)
Если вы используете Claude прямо в терминале, просто выберите пункт `4` в меню скрипта управления. Он всё настроит автоматически!

## 🧠 Как общаться с ИИ (Примеры промптов)
Просто общайтесь с ИИ как с живым ассистентом, он сам подберет нужные инструменты для работы с вашими файлами:
* *"Я оставил новые мысли в черновике `заметки_с_планерки.md`. Прочитай их и разнеси эти изменения по нужным чистовым файлам базы знаний."*
* *"Найди все файлы, где упоминается преддипломная практика, и выпиши мне все дедлайны."*

**Примечание:** Перед любым изменением файлов сервер автоматически создает резервную копию оригинала в скрытой папке `.backups`. Ваши данные в полной безопасности.
