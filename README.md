# Obsidian MCP Server

Локальный MCP-сервер для работы с базой знаний Obsidian через AI-ассистентов (Claude и другие).

## Что такое MCP

MCP (Model Context Protocol) — открытый протокол от Anthropic, который позволяет AI-моделям подключаться к внешним инструментам и данным. Вместо того чтобы копировать заметки в чат вручную, AI сам обращается к вашим файлам через MCP-сервер.

## Что делает этот сервер

Подключает вашу базу знаний Obsidian (папки с `.md` и `.txt` файлами) к Claude или другому AI-ассистенту. После настройки ИИ может:

- искать заметки по ключевым словам
- читать содержимое любого файла
- дописывать или редактировать заметки (с автоматическим созданием резервной копии)

Все операции выполняются локально — файлы никуда не загружаются.

## Инструменты (MCP Tools)

| Инструмент | Описание |
|------------|----------|
| `search_notes` | Полнотекстовый поиск по всем подключённым папкам |
| `read_note` | Чтение содержимого файла по пути |
| `safe_kb_write` | Запись в файл с автоматическим бэкапом (append / full_replace / replace_after_header) |

Перед любым изменением файла сервер автоматически создаёт резервную копию в папке `.backups`. Хранится до 50 последних бэкапов.

## Установка

Выберите инструкцию для вашей ОС:

- [Windows](README_Windows.md)
- [macOS / Linux](README_MacOS.md)

Краткий сценарий:
1. Запустить `Управление_сервером_Windows.bat` или `Управление_сервером_MacOS.command`
2. Добавить папку с базой знаний (пункт `1` в меню)
3. Сгенерировать конфигурацию для Claude (пункт `3`)
4. Вставить конфиг в `settings.json` Claude и перезапустить
